# Shopping Calculator

A Python application for comparing supermarket prices and converting between CAD and EUR.

## Features

- Discount calculator
- Package price to lb/kg/100g conversion
- Price per 100g to lb and kg converter
- CAD/lb + grams to package price calculator
- Product comparison tool
- CAD/EUR exchange rate support

## Technologies

- Python 3.14
- Git
- GitHub
- Visual Studio Code

## Usage

Run:

```bash
python supermarket_calculator_v1.py
